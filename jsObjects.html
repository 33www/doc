Document对象

属性

方法

获取元素

getElementById() 引用指定ID的对象

getElementsByName() 方法可返回带有指定名称的对象的集合

getElementsByClassName() 方法返回文档中所有指定类名的元素集合，作为 NodeList 对象

getElementsByTagName() 引用指定标签名的对象

querySelector() 获取一个CSS选择器的对象

querySelectorAll() 获取一组CSS选择器的对象

其他

write() 方法可向文档写入 HTML 表达式或 JavaScript 代码


Element对象

样式

属性

style 属性设置或返回元素的属性

classList 属性返回元素的类名

className 设置或返回元素的 class 属性

方法

classList.add() 添加类

classList.remove() 删除类

属性

  • currentStyle 行间样式（只兼容IE）
  • getComputedStyle（）获取计算后的样式（不兼容IE）
  • innerHTML 返回带标签的文本
  • innerText 返回不带标签的纯文本
  • className 设置或返回元素的 class 属性
  • id 设置或返回元素的 id
  • length 返回 NodeList 中的节点数
  • parentNode 返回元素的父节点
  • nextElementSibling 返回位于相同节点树层级的下一个节点
  • previousElementSibling 返回位于相同节点树层级的前一个元素
  • childNodes 属性返回节点的子节点集合
  • children 属性返回节点的子节点集合（非标准属性）
  • firstElementChild 返回元素的首个子节点
  • lastElementChild 返回元素的最后一个子元素
  • offsetTop 返回元素的垂直偏移位置
  • offsetLeft 返回元素的水平偏移位置

方法

  • remove() 从下拉列表中删除一个选项
  • removeChild() 从元素中移除子节点
  • appendChild() 向节点的子节点列表末尾添加新的子节点
  • insertBefore() 在已有的子节点之前插入一个新的子节点
  • setAttribute() 把指定属性设置或更改为指定值
  • this 当前对象

Windows对象

属性

Window 对象表示浏览器中打开的窗口

Document 对象使我们可以从脚本中对 HTML 页面中的所有元素进行访问

方法

alert() 方法用于显示带有一条指定消息和一个 OK 按钮的警告框

prompt() 方法用于显示可提示用户进行输入的对话框

setInterval() 方法可按照指定的周期（以毫秒计）来调用函数或计算表达式

setTimeout() 方法用于在指定的毫秒数后调用函数或计算表达式

clearInterval() 方法可取消由 setInterval() 设置的 timeout

clearTimeout() 方法可取消由 setTimeout() 方法设置的 timeout

open() 方法用于打开一个新的浏览器窗口或查找一个已命名的窗口

close() 方法用于关闭浏览器窗口
