核心

自定义属性data-ng-

表达式{{}}

指令

ng-model 指令把输入域的值绑定到应用程序变量 name

ng-bind 指令把应用程序变量 name 绑定到某个段落的 innerHTM

ng-app 声明该元素是一个AngularJS应用

练习时值可以为空

ng-init 声明变量

备注：通常在控制器ng-controller执行

ng-controller 声明函数

对象

$scope
$location
$http
$timeout
$interval

功能

把应用程序数据绑定到 HTML 元素

可以克隆和重复 HTML 元素

可以隐藏和显示 HTML 元素

可以在 HTML 元素"背后"添加代码

支持输入验证
