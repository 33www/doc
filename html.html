!DOCTYPE

!DOCTYPE 声明文档类型

HTML

备注：!DOCTYPE不属于HTML标签

备注：HTML5不基于SGML，所以不需要引用DTD

html

html 标签定义这是一个HTML文档

xmln 属性定义XML命名空间

http://www.w3.org/1999/xhtml 固定值

备注：HTML5不需要引用

head

head 标签定义文档的头部

meta 标签定义文档的元信息

charset 属性定义文档的字符编码

UTF-8 万国码

GB2312 汉字编码

备注：HTML5中代替http-equiv="Content-Type" content="text/html; charset=""；

引申：字符集

name 属性定义 content 属性的信息/值的名称

viewport

content 属性定义与 http-equiv 或 name 属性相关的元信息

width=device-width 页面宽度等于设备宽度

initial-scale=1 初始缩放值

shrink-to-fit=no 不允许用户缩放

title 标签定义文档的标题

link 标签定义文档与外部资源的关系

type 属性定义被链接文档/资源的 MIME 类型

text/css

text/javascript

href 属性定义被链接文档的位置

url 路径

rel 属性定义当前文档与被链接文档/资源之间的关系

stylesheet 表示该文档的外部样式表

icon 表示该文档的图标

media 属性定义被链接文档将被显示在什么设备上

screen 计算机屏幕（默认）

base 标签定义页面所有链接默认地址或默认目标

href 属性规定作为基准 URL 在页面中使用的 URL

url 路径

target 属性在何处打开页面上的链接

_self 默认值。在被点击时的同一框架中打开被链接文档

_blank 在新窗口中打开被链接文档

iframeName 在指定框架打开

备注：该属性会被每个链接中的 target 属性覆盖

style 标签定义 HTML 文档的样式信息

type 属性定义内容类型

text/css

script 标签定义客户端脚本

type 属性定义内容类型

text/javascript

src 属性外部脚本的 URL

body

body 此元素定义文档的主体

scrollTop 属性返回滚动的距离

通常在js中获取滚动距离时使用

布局

div 定义 HTML 文档中的分隔（division）或部分（section）

header 标签定义文档的页眉（介绍信息）

footer 标签定义 section 或 document 的页脚

aside 标签定义其所处内容之外的内容

nav 标签定义导航链接的部分

summary 定义摘要

事件属性

鼠标事件

onmousemove 属性在鼠标指针移动到元素时触发。

script 规定该onmousemove事件触发时执行的脚本。

onmouseout 属性在鼠标指针移除元素时触发。

onmouseenter 属性在鼠标指针移动到元素上时触发

不支持冒泡

onmouseleave 属性当鼠标指针离开被选元素时触发

不支持冒泡

onclick 属性在单击鼠标时触发。

onscroll 事件在元素滚动条在滚动时触发

备注：结合scrollTop条件判断 document.documentElement.scrollTop||document.body.scrollTop

windows事件

onload 属性在文档对象加载完成后触发。
