判断移动端、Android和iOS、微信浏览器

思路：

获取浏览器信息 navigator对象userAgent用户代理

通过正则匹配字符串 match(/string/i)

代码：

if (navigator.userAgent.match(/mobile/i)) { alert("Mobile"); }

if (navigator.userAgent.match(/(iphone|ipad)/i)) { alert("iOs"); }

if (navigator.userAgent.match(/android/i)) { alert("Android"); }

if (navigator.userAgent.match(/MicroMessenger/i)) { alert("微信浏览器"); }
